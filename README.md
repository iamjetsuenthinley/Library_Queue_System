# Library_Queue_System
A smart library management system that simplifies borrowing, returning, and tracking due dates.

## Book Flow Requirements

Book Flow automates core library operations in real time to make work easier for administrators and students.

- Simplify check-in and check-out workflows for books.
- Accurately track who borrowed each book, when it was issued, and when it is due.
- Track overdue items and related fines.
- Send automated reminders (SMS/email) before due dates and after overdue periods.
- Share important notices quickly, including new arrivals, schedule changes, holidays, and system alerts.
